# 元宵猜灯谜大闯关（静态版）

一个无需后端的元宵节猜灯谜小游戏，支持：

- 单人闯关（默认）
- 家庭同屏PK（可选）
- 限时答题、连击奖励、积分排行
- 趣味奖励活动卡
- 节日音效开关（浏览器本地 WebAudio）
- 题目分类筛选（按主题勾选）
- 分享战绩海报（前端 Canvas 生成）
- 本地排行榜（`localStorage`）
- 微信内打开优化（引导提示、链接复制、移动端触控适配）

## 项目结构

- `index.html`：页面结构
- `styles.css`：国潮喜庆视觉样式
- `app.js`：游戏逻辑
- `data/riddles.json`：灯谜题库（60题）
- `data/rewards.json`：趣味奖励卡

## 本地运行

因为 `fetch` 会读取本地 JSON，建议用静态服务器运行：

```bash
cd /Users/xfma/dev/codex/sandbox
python3 -m http.server 8080
```

浏览器访问：

- <http://localhost:8080>

## 游戏规则（默认单人闯关）

- 每关 5 题，正确率达到 60% 才算过关
- 每题限时可选（15/20/25 秒）
- 积分规则：
  - 答对基础 +10 分
  - 剩余时间转化为额外分
  - 每 3 连击额外奖励 +15 分
- 每通过 2 关触发一次趣味奖励卡
- 完成后可保存昵称到本地排行榜
- 最终页可一键生成战绩海报并下载分享

## 静态部署

可直接部署到 GitHub Pages / Vercel / Netlify：

1. 上传全部文件（保持目录结构）
2. 设置发布目录为项目根目录
3. 打开线上链接即可转发给家人朋友

## 微信内使用说明

1. 用户在微信里打开页面后，会看到顶部“微信内打开已优化”提示条。
2. 可点击“复制页面链接”直接复制，方便发到群聊或私聊。
3. 可点击“微信引导”查看“右上角 ···”分享步骤。
4. 战绩页支持复制战绩文案（自动附带页面链接）和海报下载。

## 纯静态限制说明

- 当前版本不接入微信 JS-SDK 签名（需要服务端签名接口）。
- 因此分享给好友时展示的是微信默认链接卡片信息。
